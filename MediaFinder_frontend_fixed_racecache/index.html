MediaFinder

Cari kemiripan video dalam 3 langkah ringan.

1) Impor video referensi. 2) Unggah kueri dan klik "Cari Mirip". 3) Lihat timeline & heatmap untuk bukti. Semua diproses lokal, aman untuk pemula.

  • 64-bit perceptual hash
  • 12-bin chroma audio
  • Late fusion scoring
  • Heatmap & DTW viewer
  • Lab robustness ffmpeg.wasm

    Bantuan & FAQ
    
    
    1) Bagaimana cara kerja?
    
    Aplikasi mengekstrak jejak visual (pHash) dan opsional jejak audio (chroma) dari video. Pencarian menghitung kemiripan per detik, menampilkan timeline/heatmap, dan menghasilkan bukti.
    
    
    2) Apakah data saya diunggah?
    
    Tidak. Semua pemrosesan berlangsung di perangkat anda (browser) menggunakan IndexedDB. Jika anda mengaktifkan Local-only, aplikasi juga menonaktifkan pemuatan library dari CDN.
    
    
    3) Kenapa export PDF/ZIP kadang tidak berfungsi?
    
    Fitur tersebut memerlukan library dari CDN (jsPDF/JSZip). Saat offline atau Local-only aktif, tombol akan menampilkan pemberitahuan.
    
    
    4) Bagaimana memasang (install) sebagai Aplikasi?
    
    Klik tombol Install di pojok kanan atas saat tersedia, atau gunakan menu browser “Add to Home Screen / Install App”.

    • Video/Audio yang diindeks disimpan lokal (IndexedDB). Anda dapat menghapus lewat tombol Reset.
